Что такое html верстка

Верстка – один из важнейших этапов создания веб-сайта, включает в себя достаточно сложный процесс формирования веб-страниц из изображений, созданных веб-дизайнером. В результате html верстки Вы получаете готовый html документ, доступный для публикации в Интернете или подключения к любой системе управления содержимым (CMS).

Какие технологии я использую

  • Верстка на HTML5 + CSS3
  • Адаптивная (сетка bootstrap), резиновая или фиксированная верстка
  • Кроссбраузерность: IE9+, Firefox, Opera, Chrome, Яндекс.Браузер
  • Соблюдение стандартов W3C (валидность)
  • Использование Pixel Perfect (точная вёрстка по макету)
  • Чистый, комментированный код
